## Installation
```shell
npx i create-react-app netflixclone
```
1. Clone repository

```shell
git clone https://github.com/akku27-cse/NETFLIX-CLONE.git
```

## Install Client (React Files)

2. Get in the React folder

```shell
~cd netflixclone
```

3. Install dependencies via npm or yarn

```shell
npm i
```

4. Start Client

```shell
npm start
```
